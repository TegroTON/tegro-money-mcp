# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and abusive
behavior are not tolerated. Maintainers may remove, edit, or reject contributions
that violate this code, and may ban contributors for inappropriate behavior.

Report unacceptable behavior to **conduct@tegro.money**. Reports are handled
confidentially.
